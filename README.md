# Data Science Salary Calculator - Excel Dashboard

An interactive Excel dashboard designed to explore and visualize global salary trends and job market data for data-related roles. This project was created for educational purposes following the "Excel for Data Analysis" curriculum by Luke Barousse.

## üìä Overview

The goal of this dashboard is to provide actionable insights into the data science job market. It allows users to filter through various roles and regions to understand compensation levels, where jobs are being posted, and the volume of opportunities available.

### Key Features
* **Dynamic Filtering:** Filter data by **Job Title** (e.g., Data Analyst, Senior Data Scientist), **Country**, and **Employment Type** (Full-time, Contractor, etc.).
* **Salary Benchmarking:** A bar chart comparing median salaries across different data roles.
* **Global Map Visualization:** A geographic heat map visualization to show data distribution across the globe.
* **Live KPI Cards:** Real-time updates for:
    * **Median Salary:** Calculated based on the selected filters.
    * **Top Job Platform:** Identifies which site (e.g., Indeed, LinkedIn) has the most listings for the selection.
    * **Job Count:** Shows the total number of data points/postings available.
